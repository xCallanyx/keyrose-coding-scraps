How to Use the Art Assignment Tool

1️⃣ Adding Students

Go to the **Manage Students** page to upload a CSV or manually add students.

2️⃣ Assignments

Use the **Manage Assignments** page to add assignments and define point values.

3️⃣ Capturing Photos

Use the **Capture Photos** page to take and store photos for student work.

4️⃣ Grading

Go to the **Grade Student Work** page to enter grades for each assignment.

5️⃣ Exporting

Use the **Export Photos** page to download photos by **Period > Assignment**.
